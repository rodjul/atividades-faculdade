Faça um programa que simule uma calculadora HP (Notação pós-fixa). O programa deve ser capaz de receber entradas do usuário que podem ser valores numéricos ou símbolos representando as 4 operações básicas (+, -, *, /).

O programa deve empilhar valores numéricos e desempilhá-los quando a entrada for uma das operações básicas e empilhar a resposta.

A resposta ao usuário se dá quando o usuário digitar "="

Exemplo: 4 <enter> 5<enter> + <enter>

Resultado = 9

DICA:

public static boolean isNumeric(String str)
{
    for (char c : str.toCharArray())
    {
        if (!Character.isDigit(c)) return false;
    }
    return true;
}
